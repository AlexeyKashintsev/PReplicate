Скрипты для PostgreSQL
----------------------
0.txt - создание БД, пользователя и схемы расположения таблиц
1.txt - создание таблиц и других объектов, необходимых для репликации
2.txt - 1 часть функций для репликации (проверить и исправить значения возвращаемых имен схем в replicate_mainSchema() и replicate_prefixSchemaName())
3.txt - 2 часть функций для репликации
4.txt - функции для логического удаления
99.txt - обертка функций для работы из схемы админа (view по всем пользователям, но без удаленных)
         (!!! Проверить и исправить имена схем по всем функциям )


Внимание
--------
1. Если необходимо только логическое удаление без репликации, то нужно установить часть используемых объектов и функций из репликации.
2. Platypus почему-то хочет DOUBLE PRECISION вместо NUMERIC параметра функции. Если перестануть работать функции репликации, то пересоздать их
   с другим типом (можно посмотреть в функциях удаления).
